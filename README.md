# TravelBus_NullClass_Project-RedBus_Elite-
TravelBus (RedBus-Elite) — A full-stack bus booking platform built with React 18, Node.js &amp; MongoDB. Features GIS route maps, Razorpay payments, tri-channel notifications, Elite Score reviews, UGC travel stories, multi-language support (5 Indian languages), OLED dark mode &amp; a complete admin dashboard.
